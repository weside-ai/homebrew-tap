# Homebrew Tap for weside.ai

Install weside CLI tools via Homebrew.

## Usage

```bash
brew tap weside-ai/tap
brew install weside
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `weside` | CLI for the weside.ai AI Companion Platform |
